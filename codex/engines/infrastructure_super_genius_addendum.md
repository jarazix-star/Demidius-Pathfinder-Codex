# Infrastructure Engine — Super Genius Addendum

## New Organizational Priority: Navigator

**Source:** *Advanced Options: Fight Like a Pirate*, PDF p. 7.

A Navigator ranger can reduce travel time or random-encounter probability, improve the bonded crew’s initiative and key awareness skills, and reroll ship-related attacks, checks, or saves. This is the clearest immediate personnel upgrade for the Dawnrunner.

## Spagyric Quartermaster Package

**Sources:** *Advanced Options: Alchemists’ Discoveries*, PDF pp. 4–6.

Recommended package:

- Spagyric Devices
- Esoteric Studies
- Mass Production
- Extra Devices, if feat economy permits
- Homunculus for workshop support and security

This package should be placed on a dedicated follower or cohort. It is strong enough to require explicit GM approval and careful handling of body-slot, maintenance, fragility, and separation rules.

## Officer Layer

The following options now form a coherent command package:

- Ship’s Officer cavalier
- Crown Privateer paladin
- Superior Tactician
- Navigator ranger
- Lookout rogue
- Order of the Griffon or Order of the Citadel for thematic officers

The organizational goal is not to duplicate Demidius’s bonuses. It is to cover different bonus types, grant teamwork feats, provide ship-wide defenses, and preserve function when Demidius is absent or incapacitated.
