# Esoteric Studies

**Rating:** A+  
**Verification:** Verified source; requires Spagyric Devices

## Analysis

Adds two spells per class level from any class list for the purpose of qualifying devices. It substantially broadens the equipment catalogue and is the necessary companion to a serious spagyric quartermaster build.
