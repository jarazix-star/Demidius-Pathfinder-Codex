# Mnemonic Vestment

**Rating:** S+ conditional

**Target:** Demidius

**Source:** *Ultimate Equipment* (PZO1123), PDF p. 217 / printed p. 216.
**Verification:** Rule text reviewed in context and the page visually checked.

Once per day, a spontaneous caster can spend a slot to cast a spell from a carried written source as though it were known. The spell must be on the caster's class list, no higher than the slot, and of the same arcane or divine type.

CR-09 lets Demidius cast wizard spells as divine spells, but it does not explicitly say that every wizard spell becomes part of his class list. Ask the GM whether CR-09 satisfies the vestment's class-list requirement. If yes, a curated spellbook makes this a remarkable 5,000 gp answer library; if no, it only broadens access within his ordinary lists.
