# Rune Hex

**Rating:** A+  
**Verification:** Verified source; acquisition unresolved  
**Source:** *Advanced Options: Witches’ Hexes*, PDF p. 5  
**Primary role:** Prepared defenses and contingencies

## Analysis

Stores a known touch-or-ranged hex in a triggered rune. This shifts a hex from active combat use into pre-positioned security, ambush, vault defense, or shipboard contingency. It is a particularly good Infrastructure option because it converts daily preparation into persistent tactical positioning.
