# Dawnrunner Siege-Crew Package

**Rating:** A

**Target:** Binky and the Dawnrunner's gunnery teams

**Source:** *Ultimate Combat* (PZO1118), PDF pp. 110, 119, 159-166 / printed pp. 109, 118, 158-165.
**Verification:** Feats, siege operation, ammunition, and corvus entries reviewed in context.

Master Siege Engineer can reduce loading and aiming to move actions; Siege Engineer grants broad siege proficiency and mishap protection; Siege Commander accelerates assembly and movement. The same source provides crew penalties, magical siege-engine rules, cannon ammunition, and the corvus boarding bridge.

Audit Binky's existing gun chemist and feat package before changing it. Master Siege Engineer is the leading throughput option, while a corvus is useful only if the campaign's vehicle and boarding rules support the Dawnrunner's tactics.
